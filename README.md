# Nashville NC Smart Traffic Demo

This is a demo web app showing how traffic light states can affect routing.

## Features
- Real map of Nashville, NC
- Real traffic light locations (OpenStreetMap)
- Simulated red/green signal cycles
- Built with Leaflet.js
- Hosted on GitHub Pages

## How to run
1. Upload files to a GitHub repository
2. Enable GitHub Pages (main branch, root)
3. Open the published URL
